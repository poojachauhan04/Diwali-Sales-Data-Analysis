# 🎉 Diwali Sales Data Analysis (Python - VS Code)

This project analyzes **Diwali sales data** using Python in VS Code.  
The focus is on **Exploratory Data Analysis (EDA)** and **data cleaning** to uncover customer behavior, product performance, and regional sales trends during the festive season.

---

## 🚀 Project Description
- **[Data Cleaning](ca://s?q=Explain_data_cleaning_in_Python_for_Diwali_sales_data)**: Handled missing values, removed duplicates, standardized formats.  
- **[Exploratory Data Analysis](ca://s?q=Explain_exploratory_data_analysis_in_Python_for_Diwali_sales_data)**: Summarized distributions, correlations, and visualized patterns.  
- **[Customer Insights](ca://s?q=Analyze_customer_insights_in_Diwali_sales_data)**: Age, gender, marital status, and occupation analysis.  
- **[Regional Trends](ca://s?q=Analyze_regional_trends_in_Diwali_sales_data)**: Top states contributing to orders and revenue.  
- **[Product Analysis](ca://s?q=Analyze_product_categories_in_Diwali_sales_data)**: Best‑selling categories and top products by sales.  

---

## 📂 Project Structure
- `data/` → Diwali Sales dataset (CSV)  
- `notebooks/` → Jupyter notebooks for EDA  
- `src/` → Python scripts for cleaning and analysis  
- `README.md` → Documentation
